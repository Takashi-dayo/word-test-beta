# みんなの単語帳（GitHub Pages版）

端末内保存専用の英単語学習PWA。アカウント、クラウドDB、アクセス解析は使用しない。

## 公開・更新手順

1. ZIPを展開する。
2. 中のファイルと `icons` フォルダを、GitHub Pages用リポジトリ直下へすべて上書きアップロードする。
3. GitHubの反映後、公開ページを再読み込みする。
4. 古い画面が残る場合はページを完全に閉じて再度開く。Service Workerのキャッシュ番号はv6.6へ更新済み。

## 主な画面

- 今日：未回答語と復習期限に達した単語を表示
- 問題：出題条件を決めてから、問題だけの画面で学習
- 登録：一つずつ登録／まとめて登録から分岐
- 全単語：検索、並べ替え、発音、編集、削除
- 分析：過去30日の追加単語数と解いた問題数を日別グラフで表示
- 設定：下部ナビゲーション右端の独立した歯車から、通知・バックアップ・復元・データ削除を操作

## 登録チェック

- 完全に同じ英単語・日本語訳は重複登録を止める。
- 同じ英単語に別の日本語訳を追加する場合は、既存の訳を表示して確認する。
- 英単語のスペルが疑わしい場合は端末内辞書から候補を表示する。
- 日本語訳を自動訂正する機能は使用しない。

## 分析データ

- 追加単語数は各単語の登録日時から集計する。
- 解いた問題数はv6.0以降の回答履歴を集計する。v6.0より前の正解日時は保存されていないため復元できない。

## 保存について

- 学習データは「ブラウザ＋正確な公開URL」ごとのIndexedDBに保存される。
- 同じURLなら既存データを引き継ぐ。
- ブラウザデータ削除、端末初期化・故障に備えてJSONバックアップを使用する。
