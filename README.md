# みんなの単語帳 v7.4（フォレスタモード基盤）

## 追加内容

- 問題画面の「問題開始」と「バックアップを書き出す」の間に「フォレスタモード」ボタンを追加。
- ボタンから「問題（フォレスタモード）」設定画面へ遷移。
- フォレスタモードでは「出題範囲」の代わりに「級・段」を表示。
- 20級〜1級、初段〜15段を選択可能。
- 出題方向・出題順・問題数の指定は通常モードと同じ。
- フォレスタ問題は `foresta-questions.js` で管理し、ユーザー登録単語とは完全に分離。
- 現在は問題データ未追加のため、開始ボタンは無効になり「問題はまだ追加されていない」と表示。
- フォレスタモードの解答結果は端末内の通常単語データ・復習履歴・分析へ記録しない設計。

## データ保護

IndexedDB名 `word-study-app-db`、ストア名 `appState`、レコードキー `main`、データバージョンは変更していない。既存の登録単語データには触れない。
