# みんなの単語帳 v7.6（全単語フォレスタモード）

## 追加内容

- 「全単語」画面右上へ「フォレスタモード」ボタンを追加。
- ボタンから「全単語（フォレスタモード）」へ遷移。
- 級・段を選択し、その範囲の単語を教材順で一覧表示。
- フォレスタ単語一覧でも英語・日本語検索と発音再生に対応。
- フォレスタ問題は閲覧専用で、編集・削除・正誤記録の対象外。
- 15級〜20級は各40問、合計240問を収録。

## データ保護

IndexedDB名 `word-study-app-db`、ストア名 `appState`、レコードキー `main`、データバージョンは変更していない。既存の登録単語データには触れない。

## 反映手順

1. ZIP内のファイルを、GitHub Pagesリポジトリ直下へ上書きする。
2. `icons` フォルダは既存のものをそのまま残す。
3. 公開ページを完全に閉じ、再度開いて再読み込みする。
4. ブラウザのサイトデータは削除しない。
