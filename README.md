# みんなの単語帳 v8.0（iPhone入力時のスクロール安定化）

## 修正内容

- iPhone純正キーボードで入力中に画面が上下へ跳ぶ問題を修正。
- 非表示の実入力欄を画面下端ではなく、表示中の回答欄と同じ位置へ配置。
- 1文字枠を横1列のスクロール表示にし、文字追加による回答欄の高さ変化を防止。
- 問題切替時の重複フォーカスを廃止し、フォーカス時のページ位置を維持。
- 正解・不正解表示後の「次の問題へ」を下部ナビゲーションの直上へ固定。
- モバイルでは結果表示後にボタンへ自動フォーカスせず、勝手なスクロールを防止。
- 日本語回答、フォレスタ問題、登録単語、学習記録の仕様は変更していない。

## データ保護

IndexedDB名 `word-study-app-db`、DBバージョン `1`、ストア名 `appState`、レコードキー `main`、アプリデータバージョン `4` は変更していない。
登録単語・学習記録・設定データを削除または初期化する処理は追加していない。

## 反映手順

1. ZIP内の全ファイルをGitHub Pagesリポジトリ直下へ上書きする。
2. 公開ページを完全に閉じ、再度開く。
3. 反映されない場合のみ、ページを再読み込みする。
4. ブラウザのサイトデータは削除しない。
