# みんなの単語帳 v7.2 — Androidアイコン修正版

## 更新方法

このフォルダ内のファイルと `icons` フォルダを、現在使用しているGitHub Pagesリポジトリの直下へ上書きする。
同じ公開URLを維持すれば、IndexedDBに保存された登録単語・回答履歴・設定はそのまま残る。

## Androidでの追加手順

1. 既存の小さいアイコンのショートカットをホーム画面から削除する。
2. Braveで公開ページを開き、一度再読み込みする。
3. アプリ内上部のインストールボタン、またはBraveメニューの「アプリをインストール」を使う。
4. 「ホーム画面に追加」しか表示されない場合は、Service Worker反映のためページを閉じて開き直す。

Braveロゴ付きのアイコンは通常のブラウザショートカットであり、PWAとしてのインストールではない。今回のmanifestにはAndroid向けmaskableアイコンを追加している。

## 変更点

- 512×512のフルブリードAndroidアイコンを追加
- 透明な角を画像内容で埋め、二重縮小を防止
- `purpose: maskable` を指定
- manifest・HTML・Service Workerのキャッシュ番号をv7.2へ更新
